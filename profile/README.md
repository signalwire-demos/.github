# SignalWire Demos

Example applications showcasing [SignalWire](https://signalwire.com) voice, video, messaging, and AI capabilities.

## Featured Demos

| Demo | Description | Try It |
|------|-------------|--------|
| [Santa](https://github.com/signalwire-demos/santa) | AI-powered Santa Claus voice assistant | [Live Demo](https://santa.signalwire.io) |
