# Support

## Demo Issues

If you're having trouble with one of our demo applications:

1. **Check the demo's README** - Most common setup issues are covered there
2. **Search existing issues** - Someone may have already reported the problem
3. **Open an issue** - Use our [issue templates](https://github.com/signalwire-demos/.github/issues/new/choose)

## SignalWire Help

## Response Times

- **Demo issues**: We aim to respond within a few business days
- **Community channels**: Often same-day responses from community members and SignalWire team

## What We Can Help With

- Bug reports for demo applications
- Setup questions for running demos locally
- Feature requests for demos
- Contributions and pull requests

## What to Ask the Community

- SignalWire API questions
- Account and billing questions
- Production deployment advice
- General SignalWire product questions
